FCF (Flow-based Chatbot Framework)

IM Bridge

用來連接通訊平台，目前包含三種 Node：Facebook In、Facebook Out、Facebook Notification

Facebook In

用來接收從 Facebook 通訊平台送來的訊息

Facebook Out

用來將訊息傳送至 Facebook 通訊平台

Facebook Notification

用來主動推播訊息至 Facebook 通訊平台

使用範例

任何於 Facebook 通訊平台輸入的訊息，會被 Facebook In Node 接收，之後會將 Message Node 裡設定的訊息藉由 Facebook Out Node 送出至 Facebook 通訊平台

當外部服務呼叫 Push Service Node 後，會將 Message Node 裡設定的訊息藉由 Facebook Out Node 傳送至 Facebook 通訊平台
