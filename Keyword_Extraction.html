Keyword Extraction

藉由串接 Api.ai 來分析使用者傳入的訊息，並擷取當中的關鍵字

使用範例

Keyword Extraction Node 會將接收到的訊息傳至 Api.ai 進行分析，並於 Api.ai 端進行設定好的關鍵字擷取，並將結果回傳至後續串接的 Node 處理。依上圖所示，後續會將關鍵字存入 Frame Node 並送至 Pull Service Node 查詢
