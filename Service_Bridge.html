FCF (Flow-based Chatbot Framework)

Service Bridge

用來連接外部服務，目前包含二種 Node：Pull Service、Push Service

Pull Service

向外部服務請求資料

Push Service

供外部服務主動推送通知

使用範例

Keyword Extraction Node 將傳進來訊息的關鍵字擷取並儲存到 Frame Node 裡，再將 Frame Node 所儲存的標準資料格式送至 Pull Service Node 所設定的外部服務進行處理，並將回傳的資料藉由 Message Node 轉為文字訊息回覆

外部服務可以藉由呼叫 Push Service Node 來傳遞資料並觸發後續串接的 Node。以上圖為例，Push Service Node 被呼叫時會將接收到的資料藉由 Message Node 轉為文字訊息送出至 Facebook 通訊平台
